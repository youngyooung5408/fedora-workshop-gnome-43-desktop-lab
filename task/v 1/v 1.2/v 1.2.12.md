## version 1.2.12

This is a new workflow feature, not a desktop bug fix.

- Add a VM-only `lab -version` command, with `lab --version` also supported.
- Report the exact saved version most recently applied through a project version launcher.
- Update every historical launcher and the future launcher generator to record only after a successful import.
- Store only the current version; timestamped history is not required.
- Do not install this command or state on the actual host machine.
