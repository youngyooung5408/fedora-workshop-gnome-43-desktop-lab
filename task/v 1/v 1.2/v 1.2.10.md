## version 1.2.10

# Safe host installation

- Keep historical version launchers as exact lab restores for comparison.
- Add a separate manifest-driven host updater that installs the latest reviewed
  project extensions without importing the full GNOME snapshot.
- Preserve host display sizing, text scaling, favorites, unrelated extensions,
  Bluetooth state, and undeclared settings.
- Preview changes, require confirmation, create a backup, and support rollback.
- Preserve and report locally customized managed settings.
